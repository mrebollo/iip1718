PROJECT TITLE: Parcial_1_1112

PURPOSE OF PROJECT:
Se está desarrollando una aplicación que usa una baraja de cartas. Para ello, se implementarán en Java las clases necesarias. Una de ellas es la clase Carta que permite representar una carta de la baraja española. La información requerida para identificar una Carta es su palo (oros, copas, espadas o bastos) y su valor (un entero entre 1 y 12).

VERSION or DATE: Noviembre 2011

HOW TO START THIS PROJECT:
Proyecto BlueJ. Abre el fichero packege.bluej, compila y ejecuta el método main de la clase JuegoCartas
