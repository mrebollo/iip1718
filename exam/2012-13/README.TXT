PROJECT TITLE: Parcial_1_1213

PURPOSE OF PROJECT:
Una aplicación para la bolsa de valores utiliza una clase para definir el valor de las acciones. Cada acción queda identificada por el nombre de la empresa y contiene cuatro valores reales: los valores de apertura, mínimo, máximo y actual de una acción. Cada acción comienza la sesión del dıa con el valor de apertura, en un momento determinado tiene un valor actual y durante la sesión ha tenido valores máximo y mínimo que han podido cambiar a lo largo de la sesión.

VERSION or DATE: Noviembre 2012

HOW TO START THIS PROJECT:
Proyecto BlueJ. Abre el fichero packege.bluej, compila y ejecuta el método main de la clase BolsaValores
